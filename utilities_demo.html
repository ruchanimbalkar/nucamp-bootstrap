Color and Spacing Utility Classes in Bootstrap

Bootstrap provides us many utility classes

We can use this classes for basic color, spacing, sizing, borders and more.

For text color we will use the text prefix classes. class= "text-value "

.text-primary

.text-secondary

.text-danger

.text-warning

.text-success

.text-info

.text-light

.text-dark

.text-white

.text-muted

Forthe background color we will use the bg prefix classes. class= "bg-value

.bg-primary

.bg-secondary

.bg-danger

.bg-warning

.bg-success

.bg-info

.bg-dark

.bg-light

.bg-white

.p-3 is for padding 3 on all sides
.pb-5 is for padding 5 on bottom
.ml-5 is for margin 5 on left

Check out more on Coloring and Spacing on the official Bootstrap website
